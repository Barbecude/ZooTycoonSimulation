# IndoZoo Tycoon – User Guide

## Capture Cage (1 Mob, Toggle)
- Klik kanan ke mob kecil untuk menangkap (maks 1 mob per cage).
- Klik kanan lagi (ke udara atau ke blok) untuk melepas mob.
- Jika lokasi pelepasan terhalang, cage tidak akan melepas dan akan menampilkan pesan error.

## Animal Tagging
- Pegang item Animal Tag lalu klik kanan pada hewan untuk membuka UI penamaan/tag.
- Untuk hewan bertubuh tinggi (mis. giraffe), bisa klik kanan ke udara (raycast) untuk menangkap target di crosshair.
- Saat tag berhasil, hewan mendapat efek partikel dan suara.
- Saat memegang Animal Tag, fungsi tag diprioritaskan dibanding “ride”.

## Toilet (Block)
- Klik kanan toilet untuk duduk.
- Jika toilet sedang dipakai pemain lain, kamu otomatis masuk antrian.
- Saat toilet kosong, antrian berikutnya akan dipanggil (jika masih berada dekat toilet).
- Pintu di sekitar toilet akan dibuka saat masuk dan ditutup kembali otomatis setelah keluar.

## Hunter (Kidnapper)
- Hunter kidnapper akan mengejar hewan bertag.
- Jika jalur terhalang barrier/enclosure, hunter akan menempatkan blok dirt sementara sebagai pijakan (non-destruktif).
